Loading methylation dataset¶

Checking metadata¶

In this tutorial we are going to use one of the methylation datasets included in the test data used for accessing model performance. Its GEO accession number is GSE152026 and basic metadata of the study can be checked at the GEO accession viewer:

Blood DNA methylation profiles from first episode psychosis patients and controls

Genome-wide patterns of DNA methylation were quantified using the Illumina Infinium HumanMethylationEPIC BeadChip (EPIC array) in DNA samples isolated from blood for schizophrenia cases, first episode psychosis patients and controls. These samples were profiled as part of a wider study where they were meta-analysed with other cohorts.

Apparently, the Illumina Infinium HumanMethylationEPIC BeadChip was used to generate methylation data, thus we expect to have around 850K measurement points for each sample.

To check basic metadata, we can download the "Series Matrix" file with:

Predicting chronological age¶

Chronological age can be predicted with either of the four models found in the model_coefficients.csv file as the appropriate linear combination of methylation values plus the intercept. More precisely, a transformation of the chronological age is predicted by the models which needs to be transformed back to the original scale with the function below:

In [78]:
def transform_prediction(pred_list, age_adult = 20):
    return [(1+age_adult)*np.exp(pred)-1 if pred < 0 else (1+age_adult)*pred+age_adult for pred in pred_list]

Here we perform the actual age predictions and append them to the metadata table:

In [84]:
# prediction with "elasticNet (239)" model
probes_to_use = list(set(df_model_coeff["probeID"]).intersection(set(beta_df["probeID"])))
coeffs_to_use = np.array(list(df_model_coeff[df_model_coeff["probeID"].isin(probes_to_use)]["elasticNet (239)"]))

meth_values_for_agepred = np.matrix(beta_df[beta_df["probeID"].isin(probes_to_use)][samples_to_keep])
predage = transform_prediction(np.array(list(df_model_coeff["elasticNet (239)"])[0] + 
                                        meth_values_for_agepred.transpose() @ coeffs_to_use).reshape(len(samples_to_keep)))
sample_metadata_df["predicted_age_elasticNet"] = predage
